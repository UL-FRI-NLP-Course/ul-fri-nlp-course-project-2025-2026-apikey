# Data Documentation

This directory stores dataset references, small local samples, and processed artifacts used for exploratory NLP work.

## Data Policy

- `raw/` is reserved for metadata files, manifests, and manually downloaded resources that are not committed in full.
- `samples/` contains tiny, version-controlled samples suitable for reproducible corpus analysis.
- `processed/` contains derived files created from scripts or notebooks and is not intended for large committed outputs.
